# 更新日志

## 2026-06-09

### AI 润色提示词优化
- 单独反馈与群反馈使用不同提示词
- 单独反馈：课堂内容→上节课复习→本节课缺点→针对行为→结果→后续措施→鼓励（7要素）
- 群反馈：课堂内容→整体情况→共同的毛病→改进措施（4要素）
- 群反馈无学生时生成整体反馈草稿，不再强制要求选择学生

### 群反馈新增课堂共性问题
- 新增"课堂共性问题"填写入口（开关控制）
- 填写内容自动追加到 AI 润色草稿中

### AI 调用方式调整
- 前端直接调用 DeepSeek API，不再依赖后端代理
- 标签数据直接使用内嵌 JSON，无需后端服务

### 项目整理
- 后端文件（`server.js`、`proxy.py`、`tags.json`、`dev-server.js`）移入 `_legacy/` 归档
- 清理 `__pycache__`、`manifest.json` 等临时文件
- 项目结构精简为纯前端单文件应用

## 2026-06-08

### 安全：API Key 移回后端代理
- 前端移除硬编码 DeepSeek API Key，改为调用后端代理
- 新增 `/api/chat` 通用聊天端点（家长沟通 AI 使用）
- `server.js` 和 `proxy.py` 两个后端同步更新
- 后端统一使用 `proxyDeepSeek` 公共函数，减少重复代码

### 数据备份：全量 JSON 导出/恢复
- 新增 💾备份 按钮：导出全部 localStorage 数据（学生、班级、反馈、模板）为 JSON 文件
- 新增 ♻️恢复 按钮：从 JSON 备份文件恢复数据（覆盖前二次确认，恢复后自动刷新）
- 导入文件支持 `.json` 格式（与 `.csv`、`.xlsx` 并列）

### 代码拆分：标签数据外部化
- 标签数据从 `index.src.html` 内嵌提取为独立 `tags.json` 文件
- `loadAllTags()` 优先从后端 `/api/tags/all` 获取完整数据，失败时 fallback 到内嵌的 3 科核心数据
- 内嵌数据保留数学/语文/英语作为离线 fallback

### 多端适配：横屏/桌面布局
- 768px+ 平板：宽度 680px，字号/间距增大
- 1024px+ 桌面：宽度 900px，内容区域居中
- 1440px+ 大屏：宽度 1100px

### AI 润色限制调整
- 移除每日次数限制和冷却时间
- 按钮简化为"🤖 AI 润色"，不再显示剩余次数

### 项目整理
- 一次性修改脚本移入 `_legacy/` 文件夹归档
- 配置 Git 仓库
- 删除开屏弹窗、联系作者入口
- `index.html` 和 `index.src.html` 不再混淆

## 2026-05-27

### 重构学段学科选择系统
- **四级学段**：幼儿园（3-6岁）、小学（6-12岁）、初中（12-15岁）、高中（15-18岁），替代原来的小学/初中/高中
- **六大分类**：文化课、艺术课、体育课、语言类、科技类、思维与素质类
- **三级下拉选择器**：学段→分类→学科，紧凑的 action-sheet 下拉替代原来的标签行
- **学段科目映射**：幼儿园暂无科目；小学11科；初中全部18科；高中9科文化课

### 其他优化
- "课前测情况"更名为"课堂测验情况"
- 新增本地开发服务器 `dev-server.js`（静态文件+API代理，支持AI润色本地测试）

## 2026-05-26

### 新增初中科学科目
选择"科学"即锁定初中段，闪光点+薄弱点按四个子学科拆分：
- **[物理]**：力学、电学、光学、热学、公式规范
- **[化学]**：化学式、方程式配平、溶解度、酸碱盐、物质检验
- **[生物]**：显微镜、光合呼吸、人体生理、遗传、生态系统
- **[地理]**：经纬网、自转公转、月相、时区、日食月食
- 每个薄弱点附带具体改进建议

### 新增魔方科目
闪光点、薄弱点标签留空，用户通过 + 按钮自行添加，框架与其他学科一致。

### DeepSeek API 安全升级
- API Key 从前端源码移除，改为服务器端代理 `/api/polish`
- Key 仅存服务器，浏览器源码不可见，杜绝泄露

### AI 润色次数调整
- 周一至周五：每天 5 次
- 周六、周日：每天 10 次

### 代码混淆压缩
- 源码保留可读版 `index.src.html` 用于编辑
- 部署时 `node build.js` 一键生成压缩混淆版 `index.html`
- 浏览器内复制代码变得难以阅读

### 群反馈修复
- 自定义标签不再出现乱码
- 多选标签全部显示，不再只显示第一条
